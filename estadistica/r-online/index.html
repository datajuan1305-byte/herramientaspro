Inicio › R en línea
⚠️ Este entorno de R corre íntegramente en tu navegador (proyecto de código abierto webR, no es RStudio completo — no tiene paneles de Entorno/Gráficos separados, pero sí edición de script y consola). Úsalo para comparar resultados con las calculadoras del sitio o con tu R/RStudio local.
🔒 Privacidad: el código y los datos que escribas aquí se ejecutan localmente en tu propio navegador — no se envían a ningún servidor nuestro ni de terceros para procesarse.

Consola de R interactiva

Escribe tu script en el panel de la izquierda del recuadro de abajo y presiona "Run" (o Ctrl+Enter) para ejecutarlo. La primera carga puede tardar unos segundos mientras el navegador descarga el motor de R.

Scripts de ejemplo

📐 Descriptiva básica
x <- c(12, 15, 18, 14, 20, 9, 16, 15, 22, 13)
summary(x)
mean(x)
sd(x)
var(x)
🔬 Prueba t (una muestra)
x <- c(12, 15, 18, 14, 20, 9, 16, 15, 22, 13)
t.test(x, mu = 15)
🔗 Correlación
x <- c(10,8,13,9,11,14,6,4,12,7,5)
y <- c(8.04,6.95,7.58,8.81,8.33,9.96,7.24,4.26,10.84,4.82,5.68)
cor(x, y, method="pearson")
cor.test(x, y)
📈 Regresión lineal
x <- c(10,8,13,9,11,14,6,4,12,7,5)
y <- c(8.04,6.95,7.58,8.81,8.33,9.96,7.24,4.26,10.84,4.82,5.68)
modelo <- lm(y ~ x)
summary(modelo)
📊 ANOVA de un factor
grupo <- factor(rep(c("A","B","C"), each=5))
valor <- c(85,90,88,92,87, 78,82,80,75,79, 92,95,91,89,94)
modelo <- aov(valor ~ grupo)
summary(modelo)
TukeyHSD(modelo)
Tip: copia el script, pégalo en el editor del recuadro, y presiona "Run". Para limpiar resultados, prueba Ctrl+L con el cursor en la consola, o simplemente recarga la página (guarda tu script aparte antes, por si acaso).
