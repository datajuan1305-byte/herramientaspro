⚠️ Este entorno de R corre íntegramente en tu navegador (proyecto de código abierto webR). Incluye editor, consola, explorador de archivos y panel de gráficos. Úsalo para comparar resultados con las calculadoras del sitio o con tu R/RStudio local.
🔒 Privacidad: el código y los datos que escribas aquí se ejecutan localmente en tu propio navegador — no se envían a ningún servidor nuestro ni de terceros para procesarse.

Entorno de R interactivo

Escribe tu script en el panel del editor y presiona "Run" (o Ctrl+Enter) para ejecutarlo. La primera carga puede tardar unos segundos mientras el navegador descarga el motor de R.
